# quantum-information
## 概要/Overview
量子アルゴリズムを量子コンピューティング用PythonライブラリであるQiskitで実装しています。  
This project implements quantum algorithms using Qiskit, a Python library for quantum computing.

## 使用言語/Language
Python,Qiskit

## 実装していきたいアルゴリズム/Algorithm
* 量子テレポーテーション/Quantum teleportation
* グローバーの探索アルゴリズム/Grover's Algorithm
* ショアのアルゴリズム/Shor's Algorithm
